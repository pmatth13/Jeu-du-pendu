# Jeu du Pendu

Projet réalisé dans le cadre de ma formation en développement web. (Believemy®)

## Description

Jeu du pendu classique où le joueur doit deviner un mot en proposant des lettres. Le joueur dispose de 6 tentatives maximum avant de perdre la partie.

## Fonctionnalités

- Clavier AZERTY interactif
- 20 mots disponibles choisis aléatoirement
- Possibilité de proposer le mot complet
- Affichage du compteur d'erreurs
- Dessin progressif du pendu en SVG
- Bouton pour recommencer une nouvelle partie

## Technologies utilisées

- HTML5
- CSS3
- JavaScript

## Comment jouer

1. Ouvrir le fichier index.html dans un navigateur
2. Cliquer sur les lettres du clavier pour proposer une lettre
3. Ou taper un mot complet dans le champ de saisie et cliquer sur "Valider"
4. Le jeu se termine après 6 erreurs ou lorsque le mot est trouvé

## Structure des fichiers

- index.html : structure de la page
- style.css : mise en forme et responsive
- app.js : logique du jeu
